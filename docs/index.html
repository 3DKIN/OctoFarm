OctoFarm

Web server and client combo for managing and monitoring multiple Octoprint instances.
Created and maintained by James Mackay from NotExpectedYet

What is OctoFarm?

OctoFarm is an easy to setup and install web server and client for unifying multiple instances of Octoprint. You can manage and monitor as many instances as you want from a single interface giving you full control over your 3D printer farm.

Management

Manage and control every aspect of your octoprint run farm.

Monitoring

With a selection of views that enable full farm monitoring.

History

Keep track of historical successful, failed or cancelled prints.
